# TrustGrant Ports

Backend-agnostic port traits for the [TrustGrant](https://github.com/STEXS-Technologies/trustgrant) delegation protocol.

Trait-based interfaces for storage backends, authority discovery sources,
and revocation sources. Implement these traits to plug TrustGrant into
your infrastructure.

See the [trustgrant] crate for full documentation and integration guide.
