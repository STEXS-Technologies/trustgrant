# TrustGrant Issue

Draft and signable document construction for the [TrustGrant](https://github.com/STEXS-Technologies/trustgrant) delegation protocol.

Provides builders for constructing grants (drafts with authorities, resource scopes,
audience scopes, capabilities, constraints, revocation policy) and finalizing
them into signed documents.

See the [trustgrant] crate for full documentation and integration guide.
