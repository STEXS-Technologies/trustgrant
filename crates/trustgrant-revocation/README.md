# TrustGrant Revocation

Revocation state management and proof verification for the [TrustGrant](https://github.com/STEXS-Technologies/trustgrant) delegation protocol.

Provides revocation proof document types, status tracking (revoked/active),
freshness checking, revocation source kind classification, and
serialization for verified revocation records.

See the [trustgrant] crate for full documentation and integration guide.
